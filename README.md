# 生成markdown链接文本插件

Github：[xwjie/chrome-copy-markdown-link-plug](https://github.com/xwjie/chrome-copy-markdown-link-plug)

## 安装

`扩展程序`里面打开`开发者模式`，点击`加载已解压的扩展程序`, 直接选择 `source` 目录安装。

点击右键：

![效果](snap.png)

结果：

```
[xwjie (晓风轻)](https://github.com/xwjie)
```
